8. `SNMP`
: Simple Network Management Protocol
네트워크 관리자가 네트워크 성능을 관리하고 네트워크 문제점을 찾아 수정하는데 도움을 주는 응용 계층의 프로토콜.

9. `RIP`

- SMB : server message block - LAN이나 컴퓨터 간의 통신에서 데이터 송수신 프로토콜.
- OSPF : open shortest path first - 최단 경로 우선 프로토콜.

- UDP : 단방향 전송, 비신뢰성, 한꺼번에 많은 양 전송(TFTP), 수신 측 인증 불필요.

: 시스템에서 사용 가능한 메모리 양을 늘리기 위해 디스크 장치를 이용하는 것.
